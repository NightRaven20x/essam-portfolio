# Essam Alarbashli | Professional Web Portfolio 🚀

> **Live Deployment:** [View the Live Site Here](https://NightRaven20x.github.io/essam-portfolio/)

A high-performance, multi-page portfolio engineered from scratch to showcase my work in Front-End Web Architecture, AI Logic Integration, and Event Operations. Built with a strict "Design Beyond Extent" dark-mode aesthetic.

---

## 🏗️ Technical Architecture
This platform was built without the use of heavy frameworks or pre-made templates. The goal was to demonstrate mastery of core web technologies, responsive design principles, and lightweight micro-interactions.

* **Frontend Stack:** HTML5, CSS3, Vanilla JavaScript.
* **Layouts:** Advanced implementation of CSS Flexbox and CSS Grid.
* **Backend / Routing:** Serverless form endpoint via [Formspree](https://formspree.io/) with custom success redirects.
* **Deployment:** CI/CD pipeline via GitHub Pages.

---

## ✨ Core Features
* **Full Multi-Page Ecosystem:** 8 interconnected pages including a structured Blog, detailed Project case studies, and an automated FAQ.
* **Responsive "Mobile-First" Design:** Fluid typography and strict media queries ensure flawless rendering across ultra-wide monitors, tablets, and mobile devices.
* **Serverless Contact System:** Complex HTML forms utilizing dropdowns, radio buttons, and hidden inputs, securely routed to a private email inbox.
* **Custom Micro-Interactions:** * Sliding off-canvas mobile hamburger menu with CSS transform animations.
  * Pure CSS `@keyframes` Easter eggs (e.g., the interactive camera flash).
  * Global custom `crosshair` cursor integration for premium agency feel.
