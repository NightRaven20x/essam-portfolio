HIKING SEEKERS - AGENCY

A premium travel booking platform for Algerian mountain expeditions. I developed this responsive, high-performance web application using Next.js and TypeScript to architect a scalable, component-driven UI with dynamic routing. The frontend is styled entirely with the new Tailwind CSS v4 engine, leveraging native CSS variables and backdrop-filter utilities to create custom glassmorphism effects and viewport-scaled editorial typography.
