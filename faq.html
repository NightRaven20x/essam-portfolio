What technologies do you specialize in?

I am an AI engineering student and Front-End Web Engineer. My core stack includes React, Next.js, TypeScript, JavaScript, and HTML5/CSS3 for the frontend. For backend and lower-level logic, I frequently develop in Python and C. I am also highly experienced with design tools like Figma and modern styling engines like Tailwind CSS v4.

Are you available for freelance projects?

Yes. I am currently taking on select web engineering and brand identity projects. If you need a high-performance website, an interactive digital experience, or a scalable UI architecture, please use the "Book Appointment" form below to reach out with your project details.

What is your design-to-development process?

Everything starts with a strategic wireframe and a pixel-perfect mockup in Figma to establish the brand aesthetic. Once the layout is approved, I transition to code, building a component-driven architecture using React or Next.js. I prioritize mobile responsiveness, W3C accessibility standards, and clean, modular CSS from day one.

Do you only do web design, or do you handle logic?

I handle deep technical logic. Beyond front-end layouts, I have built machine learning pipelines using Python, Flask, and scikit-learn (such as the Mind-Journey project), and programmed complex, low-level algorithms like hospital triage simulations in C. I bridge the gap between creative design and heavy technical execution.
